# qr_code_js
# 📱 E-commerce QR Code Generator

Sistema desenvolvido em **Node.js** para geração rápida de QR Codes via terminal. Ideal para times de marketing e e-commerce que precisam gerar links de produtos em massa para materiais impressos ou redes sociais.

## ✨ Diferenciais
- **Cores Customizadas**: QR Codes gerados com as cores da marca.
- **FS Management**: Criação automática de diretórios de saída.
- **Async Workflow**: Processamento assíncrono para garantir performance.

## 🛠️ Como usar
1. Instale as dependências: `npm install`
2. Inicie o gerador: `node src/index.js`
3. Insira a URL e o nome do arquivo.
4. Verifique a pasta `/output`.
